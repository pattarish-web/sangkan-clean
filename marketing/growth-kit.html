เช็คลิสต์ Google Search Console

  1. เปิด Search Console ด้วยบัญชี Google ที่ยืนยันโดเมนแล้ว (เว็บมี meta verification อยู่แล้ว)
  2. เลือกทรัพย์สิน www.sangkanclean.com
  3. เมนูซ้าย → Sitemaps
  4. ใส่ sitemap.xml แล้วกด Submit
  5. ตรวจว่าสถานะเป็น Success และเห็น sitemap-pages.xml + sitemap-blog.xml
  6. เมนู Pages / การจัดทำดัชนี — ดูว่าหน้าสำคัญถูก Index
  7. (ทางเลือก) ใช้ URL Inspection กับหน้าแรกและหน้า landing หลัก แล้วกด Request indexing

  1. เปิด business.google.com → สร้างหรือจัดการธุรกิจ
  2. ชื่อ: Sangkan Clean (หรือชื่อที่ตรงกับป้าย/เอกสาร)
  3. หมวดหมู่หลัก: บริการทำความสะอาด / Cleaning service
  4. ใส่เบอร์ 063-686-5134, เว็บ https://www.sangkanclean.com/, เวลาเปิด 08:00–18:00 ทุกวัน
  5. คำอธิบายสั้น: บริการ Big Cleaning แม่บ้านประจำ หลังก่อสร้าง ทั่วกรุงเทพฯ และปริมณฑล

  7. ยืนยันสถานที่ตามที่ Google ขอ (ไปรษณีย์ / โทร / อีเมล)
